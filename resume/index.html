❯ cat projects // 核心项目 — 点击标题折叠/展开
AI 智能体教练 ▾ 2025.09 — 12
# 背景
企业销售、客服等岗位的软技能培训（异议处理、沟通谈判）长期依赖线下角色扮演，成本高、无法规模化。
# 方案
  • 设计陪练 + 教练双智能体解耦架构：陪练模拟客户/谈判对手，教练负责评估反馈，对话状态机管控全程演练节奏
  • 搭建系统级 / 任务级 / 动态级三层 Prompt 模板库，配套硬规则校验兜底；通过多版本 A/B 测试持续收敛智能体人设
  • 基于 Dify 编排对话生成、规则校验、评估全链路工作流
  • 构建规则 + 语义双轨评估体系，陪练结束后自动输出结构化能力诊断
# 结果
落地 12 个标准化培训场景，覆盖多类岗位核心技能，支撑企业软技能培训规模化迁移
平台 AI 知识库建设 ▾ 2024.12 — 2025.10
# 背景
AI 助手、AI 陪练等场景共用私域知识，此前各场景独立维护，内容重复、召回质量参差不齐。
# 方案
  • 主导 RAG 全链路产品设计，覆盖文档解析、智能切片、自动打标、向量构建到检索问答的完整闭环
  • 推动密集向量 + 稀疏向量双模式落地，语义理解与关键词精准匹配互补
  • 设计多路召回 + 融合排序 + 阈值过滤策略，在召回率与准确率之间找平衡
  • 完成 PDF、Word、视频等多格式知识的结构化处理，一次入库、多 AI 场景复用
# 结果
建成统一私域知识底座，支撑 3 条产品线，解决「问不到、答不准」的核心体验问题
AI 助手 ▾ 2024.12 — 2025.03
# 背景
企业学习平台资源分散，内部知识无法被通用 AI 调用，学习与内容创作效率低。
# 方案
  • 设计意图识别 → 能力路由 → 工具调用完整工作流，串联各功能模块
  • 搭建 RAG 检索模块，打通知识库、课程库与向量检索，实现私域知识精准问答
  • 设计专家广场人机协同机制：AI 无法解答时自动转接平台专家
  • 按学习场景拆分 Prompt 模板，实现教案、文案、PRD 等内容的结构化生成
# 结果
上线后成为平台知识消费的主要入口，用户可在对话中直接完成知识检索、内容生成与任务操作，减少多系统间跳转
